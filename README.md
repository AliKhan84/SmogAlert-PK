# SmogAlert PK - Air Quality Monitoring & Prediction System

## Project Structure

```
SmogAlert-PK/
├── data/           # Raw and processed air quality datasets
├── models/         # Trained ML models and model artifacts
├── src/            # Core Python modules for data processing and modeling
├── dashboard/      # Streamlit web application for visualization
├── outputs/        # Generated reports, plots, and predictions
└── requirements.txt # Python dependencies
```

### Folder Descriptions

- **data/**: Stores raw air quality data (CSV, JSON) and preprocessed datasets ready for modeling
- **models/**: Contains saved machine learning models (joblib, h5, pkl files) for predictions
- **src/**: Core application logic including preprocessing, model training, and alert generation
- **dashboard/**: Interactive Streamlit dashboard for real-time monitoring and visualization
- **outputs/**: Generated visualizations, prediction reports, and alert logs
